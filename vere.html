Project Overview 项目概览

Vere is a narrative-driven experimental platformer created during a 42-hour game jam under the theme "Merge." The game takes place inside a corrupted video game stored on a forgotten floppy disk. As the digital world decays, fragments of the game's interface, unused assets, and broken systems begin to merge into the environment. Players navigate a collapsing digital landscape where the boundaries between game interface and world space no longer exist. UI elements become platforms, dialogue windows appear as physical obstacles, and fragments of the game's systems drift through space as environmental hazards.

Vere 是一款在 42 小时 Game Jam 中、以「Merge」为主题创作的叙事向实验平台游戏。故事发生在一张被遗忘软盘中存储的、已损坏的游戏内部；随着数字世界腐朽，游戏界面碎片、未使用的资源与崩坏的系统开始与环境融合。玩家穿行于正在崩塌的数字景观中，游戏界面与世界空间的边界已不复存在。UI 元素变成平台，对话窗口成为实体障碍，游戏系统的碎片在空间中飘荡，化作环境危险。

Design Pillars 设计支柱

1

Interface Becomes Environment界面即环境

The central design idea is that elements normally outside the game world become part of the physical space. UI components such as health bars, dialogue boxes, and interface frames transform into: moving platforms, environmental hazards, navigational structures.核心设计理念是：通常位于游戏世界之外的元素成为物理空间的一部分。血条、对话框、界面框等 UI 组件转化为：移动平台、环境危险、导航结构。

This mechanic creates a meta-game experience, where players explore the internal structure of the game itself.这一机制创造出元游戏体验，玩家探索的是游戏自身的内部结构。

2

Glitch as Level Design故障即关卡设计

Instead of traditional obstacles, the game world is built from digital corruption and merging systems. Broken UI fragments, drifting objects, and impossible geometry communicate that the game world is no longer stable.游戏世界由数字崩坏与融合系统构成，而非传统障碍。破碎的 UI 碎片、漂浮的物体与不可能几何传达出游戏世界已不再稳定。

This supports both the visual identity and the narrative theme of a collapsing digital space.这同时支撑了视觉身份与崩塌数字空间的叙事主题。

3

Narrative Through Environment通过环境叙事

Rather than relying heavily on exposition, the story is conveyed through environmental design. The merging of UI elements and world objects reflects the game's narrative premise: the boundaries between system, interface, and world are dissolving.故事通过环境设计传达，而非依赖大量说明。UI 元素与世界物体的融合呼应叙事前提：系统、界面与世界的边界正在消解。

The player gradually realizes that Vere may not be escaping the world—but instead merging into a new reality beyond it.玩家逐渐意识到，Vere 或许不是在逃离世界，而是在融入其之外的新现实。

Core Mechanic: UI Environmentalization核心机制：UI 环境化

The core mechanic transforms traditional UI elements into gameplay structures. Examples include: Health bars functioning as moving platforms; Dialogue windows acting as navigation gates; Interface frames forming environmental boundaries.核心机制将传统 UI 元素转化为玩法结构。例如：血条充当移动平台；对话窗口充当导航门；界面框构成环境边界。

This mechanic reinforces the theme "Merge" by blending two normally separate layers: interface layer and world layer. Players are forced to interact with elements that normally exist outside the playable space.该机制通过融合两个通常分离的层面——界面层与世界层——强化「Merge」主题。玩家不得不与通常存在于可玩空间之外的元素互动。

Level Design Approach关卡设计思路

Because the project was created within a 42-hour development window, the level design focused on modular interactions. Instead of building entirely new mechanics for each stage, we reused the core UI-environment mechanic in different contexts.项目在 42 小时开发窗口内完成，关卡设计聚焦模块化互动。我们在不同情境下复用核心 UI-环境机制，而非为每一阶段构建全新机制。

Examples: platforming sequences using floating UI panels; navigation puzzles using interface frames; environmental hazards made from corrupted UI fragments. This approach allowed us to maintain design coherence while working under tight time constraints.例如：用悬浮 UI 面板做平台序列；用界面框做导航谜题；用损坏的 UI 碎片做环境危险。这一思路使我们在时间紧张下仍保持设计一致性。

Player Experience Goals玩家体验目标

The intended player experience focuses on three elements: Disorientation—players should feel that the world itself is unstable and unfamiliar. Discovery—by interacting with UI structures in unexpected ways, players gradually understand the logic of the merged world. Narrative Curiosity—environmental storytelling encourages players to question what the world actually is.预期体验聚焦三点：迷失感——世界本身应显得不稳定、陌生。发现——通过非常规方式与 UI 结构互动，玩家逐渐理解融合世界的逻辑。叙事好奇——环境叙事促使玩家追问：这个世界究竟是什么。

Is Vere escaping a broken game… or entering a new reality created from its fragments?Vere 是在逃离一个崩坏的游戏……还是进入由其碎片构成的新现实？

Design Challenges设计挑战

The primary challenge was balancing ambitious ideas with limited development time. The UI-environment concept had many possible variations, but implementing too many would have exceeded the scope of a 42-hour jam.主要挑战是在有限开发时间内平衡宏大想法。UI-环境概念有多种可能变体，但实现过多会超出 42 小时 Jam 的范围。

To address this, the team focused on: one strong core mechanic; reusable interaction systems; visually distinctive environments. This allowed the game to deliver a clear concept without overextending development resources.为此团队聚焦：一个强有力的核心机制；可复用的交互系统；视觉上鲜明的环境。使游戏在不过度消耗资源的前提下传达清晰概念。

Reflection反思

This project demonstrated the importance of strong central mechanics in rapid development environments. A clearly defined design concept helped the team align quickly despite the limited time frame.本项目体现了在快速开发环境中强有力核心机制的重要性。清晰的设计概念帮助团队在有限时间内快速对齐。

Key lessons: Game jam projects benefit from a single strong mechanic rather than many small systems; Environmental storytelling can reinforce mechanical ideas; Mechanic reuse across levels helps maintain scope control.主要收获：Game Jam 项目更适合一个强核心机制而非多个小系统；环境叙事能强化机制理念；机制在关卡间复用有助于控制范围。

If given more development time, I would expand the system further by introducing: interactive menu systems as puzzle mechanics; corrupted save-file environments; dynamic UI-based hazards that respond to player actions. These additions could deepen the idea of exploring the internal structure of a game world.若有更多开发时间，我会进一步拓展系统：将交互菜单系统作为谜题机制；损坏的存档文件环境；根据玩家行动变化的动态 UI 危险。这些扩展能深化「探索游戏世界内部结构」这一理念。

The Glitch Reality 故障现实

Inside, the game world is falling apart. Fragments of flat UI and all the things that were never meant to exist and hurt anyone within the world merged and drift aimlessly through space. 在内部，游戏世界正在分崩离析。扁平UI的碎片以及所有本不该存在且会伤害世界内任何事物的物体融合在一起，在空间中漫无目的地漂浮。

Vere must cross countless obstacles and get through relentless trials to find her way home… or so she believes. But beyond the flickering chaos of flying glitches everywhere lies something far greater than she ever imagined. The truth. Vere必须跨越无数障碍，通过无情的试炼才能找到回家的路……至少她是这么认为的。但在到处闪烁的故障混乱之外，存在着比她想象中更伟大的东西。真相。

Vere… welcome merging into a new world. Vere……欢迎融入新世界。
