# Game Designer Portfolio (Static Version)

This is a static HTML/CSS version of your portfolio website. It does not require Node.js, React, or any build tools.

## How to use

1.  Simply double-click `index.html` to open it in your browser.
2.  That's it!

## Editing content

To change text or images, open `index.html` in any text editor (like Notepad, TextEdit, or VS Code) and edit the text directly.

## Customization

-   **Colors**: defined in the `<script>` tag at the top of the file under `tailwind.config`.
-   **Images**: Search for `img src="..."` or `video src="..."` to replace media.
-   **Text**: Search for the text you want to change (e.g., "YOUR NAME") and replace it.
